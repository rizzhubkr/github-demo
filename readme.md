## `Code` & Code Blocks

### Inline `code`

Use backticks `` ` `` for inline `code`.

### Multi-language Code Blocks

#### JavaScript
```js
// Greet function
function greet(name) {
  return `Hello, ${name}! 👋`;
}
console.log(greet("World"));
```

#### Python
```python
# Greet function
def greet(name):
    return f"Hello, {name}! 👋"

print(greet("World"))
```
